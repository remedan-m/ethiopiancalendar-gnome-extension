# Ethiopian Calendar GNOME Shell Extension

A GNOME Shell extension that displays the Ethiopian calendar date in the top panel, alongside options for customizing the display format, showing the day of the week, and positioning the panel indicator.

---

## Features

- Shows the current date in the Ethiopian calendar format in the GNOME top panel.
- Supports multiple Ethiopian date display formats.
- Option to show/hide the day of the week in Amharic.
- Allows adjusting the position of the indicator in the panel (Far Left, Left, Center, Right, Far Right).
- Live updates the date every minute.
- User-friendly popup menu to configure date format, position, and show day.
- Preferences accessible from the popup menu.

---

## Installation

1. Clone or download this extension into your local GNOME Shell extensions folder:

2. Make sure the directory contains the following files:
- `extension.js`
- `metadata.json`
- (Other necessary files if any, e.g., schema files)

3. Restart GNOME Shell (Press `Alt+F2`, type `r` and Enter) or log out and back in.

4. Enable the extension using gnome extension command or GNOME Tweaks or the Extensions app.

---

## Usage

- Click the Ethiopian Calendar indicator in the top panel to open the popup menu.
- Configure the panel position submenu to set where the indicator appears.
- Change the date format from the dropdown menu to customize the date display.
- Toggle "Show Day of Week" to display the Amharic weekday before the date.
- Access Preferences from the popup menu for further settings if implemented.

---

### Ethiopian calendar conversion

The extension converts the current Gregorian date to the Ethiopian calendar date internally.

### Panel placement

The extension supports multiple positioning options in the GNOME Shell panel.
